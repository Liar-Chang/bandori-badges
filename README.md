## ✨ 功能 / Features

- 🔍 多條件篩選（樂團、角色、發售年、活動、尺寸、類型）
- 🌏 四語言切換
- ⭐ Google 登入後可使用：
  - 我的最愛清單（手動分類）
  - 智慧清單（依條件動態追蹤）
  - 蒐集記錄（日期、數量、備註）
  - 批次選取
- 📱 手機版完整支援
- 🎵 推薦歌曲（Spotify 嵌入）
